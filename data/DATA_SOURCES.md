# Parai beat training data — where it comes from

## Class layout (`data/parai/`)

Audio is organized by **stick / stroke type** (aligned with `data/labels.json`):

| Folder | Parai notes (examples) | Meaning |
|----------|-------------------------|----------------|
| `big/`   | Thi, Ku, Ka             | Adikuchi-style |
| `small/` | Tha | Sundukuchi |
| `both/`  | Theem                   | Both sticks    |

## What you have in this repo today

1. **Bootstrap (synthetic)**  
   Running `python model_training/bootstrap_parai_data.py` fills `data/parai/*` by:
   - Copying the **placeholder tones** already in `public/audio/` (generated by `npm run generate:audio` — simple sine-wave WAVs, not real Parai recordings).
   - Adding **light augmentations** (pitch shift, noise) so fine-tuning has more than a handful of files.

   **Source:** entirely inside this project (no external download).

2. **Real Parai audio (what you should add for production quality)**  
   Record your own WAV/MP3 clips (0.5–3 s per hit) and place them in the right class folder. Public, labeled Parai stroke datasets are **not** common; community practice is local recording or curated teaching material with permission.

3. **Optional proxy data (not included automatically)**  
   For experimentation only, some teams use **CC-licensed percussion** from [Freesound](https://freesound.org) (filter by license). That is **not** Parai-specific; use only if license terms allow and you relabel carefully.

## Pretrained model (not “data” but source of weights)

Fine-tuning uses **AST fine-tuned on AudioSet** from Hugging Face:

- **Model ID:** `MIT/ast-finetuned-audioset-10-10-0.4593`  
- **Paper / family:** Audio Spectrogram Transformer (AST); pretrained on Google **AudioSet** (large-scale YouTube-labeled clips, including many percussion-related classes).  
- **Download:** `python model_training/download_pretrained.py` caches weights under `model_training/pretrained_checkpoints/` (or Hugging Face cache).

This gives a strong audio backbone; your Parai classes replace only the classification head during fine-tuning.
